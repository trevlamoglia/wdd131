MakeBakeTake Project Plan Trevor Lamoglia WDD 131 Overview Purpose The MakeBakeTake website provides an interactive platform where users can browse handmade clay keychains, view product details, add products to a cart counter, and submit a custom order request. Audience Teenagers, young adults, and anyone interested in handmade custom clay keychains and personalized gifts. Dynamic elements - Featured product preview updates when a user selects a keychain. - Product cards are dynamically displayed using JavaScript. - Add to Cart button updates the cart counter. - Custom order form collects the user's name, email, selected design, quantity, and design details. - Form validation checks that required fields are completed before submission.
